Comment Voting Plugin for Hotaru CMS
---------------------------------
Created by: Nick Ramsay

Description
-----------
Let users vote on comments. Both up and down votes are displayed together.

Instructions
------------
1. Upload the "comment_voting" folder to your plugins folder.
2. Install it from Plugin Management in Admin.
