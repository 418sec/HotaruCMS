Comments Widget for Hotaru CMS
--------------------------------
Created by: Nick Ramsay

Description
-----------
Displays latest comments in a widget block.

Instructions
------------
1. Upload the "comments_widget" folder to your plugins folder. Install it from Plugin Management in Admin.
